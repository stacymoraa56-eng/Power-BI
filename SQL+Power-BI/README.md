# SQL + Power BI Sales Analytics Dashboard

## Project Overview
This project analyzes sales data using SQL and visualizes insights using Power BI.  
The goal is to transform raw transactional data into actionable business insights.

---

## Tools Used
- SQL (Data extraction, cleaning, aggregation)
- Power BI (Dashboard visualization)

---

## Project Workflow
1. Data cleaning and transformation performed using SQL
2. Key business metrics computed (sales, revenue, product performance)
3. Data imported into Power BI
4. An interactive dashboard created for insights

---

## Key Insights
- Top performing products identified
- Monthly sales trends analyzed
- Revenue distribution by category visualized

---

## Files in this Repository
- `sql/queries.sql` → SQL scripts used for analysis
- `powerbi/sales_dashboard.pbix` → Power BI dashboard file
- `images/` → Dashboard screenshot
